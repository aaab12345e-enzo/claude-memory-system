> 示例·反馈类记忆:这是"记录你的用户偏好"的样例,换成你自己用户的实际偏好。

用户团队日常用聊天工具沟通(短消息、对话式)。给团队成员看的东西,要做成能直接贴进聊天的**短、轻、口语化**形式,不要带表格 / emoji 等级标 / `file:line` 满天飞 / 多章节的长 markdown 文档——不同受众读着都不舒服(非技术方嫌太技术,技术方嫌太啰嗦)。

**Why:** 沟通载体是聊天不是文档库;受众可能分不同角色(有的要业务语言、有的要精炼技术点),长正式文档不匹配任何一方。

**How to apply:** 默认给"能直接转发群里"的几句话,不主动开大文档。要分受众时各给一条短的:业务方给业务话、技术方给关键改动+在哪。需要存档时再落一份文档,但也尽量精简。相关:[[vscode-chat-links-dont-open]](引用文件给路径别让点链接)。
