# Validation scripts (optional)

A place for small scripts that maintain the memory library. A typical one: **"logic map vs. code" health check** — batch-check whether the file paths / functions referenced in each module map still exist and whether the logic is still correct (a wrong map is the most fatal: it steers the AI onto code that doesn't exist or has been deprecated).

Run it once during a daily consolidation; fix any mismatch on the spot.

> Empty directory, add to it as needed.
